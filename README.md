# post-api

It's a service for posting articles about different programming languages:

- JAVA;
- PYTHON;
- JAVASCRIPT;
- C;
- KOTLIN;
- BASH;
- BAT. 

Stacks that were used:
- Spring Boot;
- Gradle;
- Travis CI;
- Swagger;
- MongoDB;
- Mongobee;
- Spring Security;
- Spring Data;
- Lombok;
- JUnit;
- DevTools;
- ModelMapper;
- Checkstyle;



For use:
- gradle bootRun
- gradle build
- java -jar 1.0.SNAPSHOT.jar
